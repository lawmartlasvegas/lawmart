# LawMart Premium GitHub Pages Website

Upload the contents of this folder directly to the root of your GitHub repository:

- `index.html`
- `styles.css`
- `script.js`
- `assets/` folder

In GitHub, go to **Settings → Pages**, choose **Deploy from a branch**, select **main**, and select **/(root)**.

Before publishing, replace the placeholder phone number and email address in `index.html`. The contact form currently uses FormSubmit and will require email confirmation the first time it is used.

LawMart should have all advertising copy and Nevada document preparation disclosures reviewed for applicable registration and compliance requirements.
