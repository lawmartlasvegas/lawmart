# LawMart GitHub Pages Website

A responsive, single-page static website for LawMart, a Las Vegas document preparation company.

## Before publishing

1. Replace the placeholder phone number `(702) 555-0100` in `index.html`.
2. Replace `hello@lawmartnv.com` with the real email address.
3. Create a form at Formspree.io and replace `YOUR_FORM_ID` in the contact form action.
4. Have Nevada counsel or a qualified compliance professional review all service descriptions and disclaimers.
5. Add any required Nevada document preparation registration number, business license details, physical address, and mandated disclosures.

## Publish with GitHub Pages

1. Create a new GitHub repository, such as `lawmart-website`.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`, then click **Save**.
6. GitHub will provide the public site URL after deployment.

## Custom domain

In **Settings → Pages**, add your domain under **Custom domain**. Then update your domain provider’s DNS settings using GitHub’s current instructions.

## Notes

- No build step or framework is required.
- The site uses Google Fonts. For a fully self-hosted site, replace them with locally hosted font files or system fonts.
- The contact form does not work until a valid Formspree ID is added.
