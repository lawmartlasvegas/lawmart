Checklist administrativo para ajuste de status

Um roteiro em português para reunir informações antes da preparação dos formulários escolhidos para ajuste de status.

Conteúdo informativo: não é aconselhamento jurídico e não determina elegibilidade.

Ajuste de status é um processo jurídico com requisitos que variam. Este checklist não determina se alguém pode solicitar o benefício; ele serve apenas para organizar informações depois que o cliente já identificou o serviço apropriado.

Documentos pessoais

Reúna documentos de identidade, registros civis e cópias legíveis dos documentos migratórios relevantes.

Histórico de entradas e endereços

Organize datas, locais, endereços e outras informações históricas solicitadas nos formulários.

Informações familiares

Prepare nomes completos, datas e relacionamentos conforme solicitado.

Documentos de apoio

Use uma lista de controle para marcar cada anexo exigido nas instruções atuais do formulário.

Revisão independente

Antes de assinar, confira todas as respostas. Dúvidas jurídicas devem ser revisadas com profissional autorizado.

Importante

Use sempre as instruções governamentais vigentes para o formulário escolhido. Para análise jurídica, procure um advogado licenciado ou representante credenciado.
