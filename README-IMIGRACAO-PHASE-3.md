# Imigração Brasileira — Phase 3

Adds a Portuguese resource center, five evergreen SEO articles, conversion-event hooks, Google Business integration point, Formspree-compatible form handling, and a reviews-ready section without fabricated testimonials.

## Optional configuration
Edit `site-config.js`:
- `GA4_MEASUREMENT_ID` for analytics
- `IMIGRACAO_GOOGLE_BUSINESS_URL` for the exact Google profile
- `IMIGRACAO_FORM_ENDPOINT` for Formspree or a compatible endpoint

Blank form endpoint keeps the email-app fallback.
